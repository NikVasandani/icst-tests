Instructions

This is a test of your ability to see differences in figures.

Each problem consists of one reference card on the left and eight comparison cards on the right. Decide whether each comparison card is the same as or different from the reference card.

Same = rotated only (can be slid into position):
→ S
Different = flipped / mirrored:
→ D

Mark S if the card is the same (rotated only). Mark D if the card is different (flipped/mirrored). Click again to deselect.

You have 3 minutes. Work as quickly as you can without sacrificing accuracy. Your score = correct − incorrect, so guessing at random will not help.
