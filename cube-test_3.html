Cube Comparisons Test — Instructions

How It Works

Each question shows two drawings of cubes. Each cube has 6 faces with a different letter, number, or symbol on each face. No symbol appears more than once on a given cube.

Your task is to decide whether the two drawings could represent the same cube viewed from different angles, or whether they must be different cubes.

Example — SAME Cube

If cube A is rotated so the × on top is replaced by ○, and B (which was hidden) becomes visible as C — these could be the same cube turned on its side. The key is that no visible face contradicts the first cube's arrangement.

Example — DIFFERENT Cubes

If the A on the left cube faces you with N on top, then N should still be adjacent to A on the right cube — but N is nowhere to be seen, replaced by symbols that contradict the first cube. These must be different cubes.

⚠ Scoring: Your score = correct answers − incorrect answers. Guessing randomly will not help. Only answer if you have some idea which choice is correct. Unanswered questions do not count against you.
⏱ Timing: In test mode, you have 3 minutes for 60 trials. Work quickly but accurately. In practice mode, there is no time limit and you get feedback.
⌨ Keyboard shortcuts: Press S for Same, D for Different.
